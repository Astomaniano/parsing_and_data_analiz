Проект: Parsing and Analiz

Описание:

Этот проект парсит данные о стоимости квартир с помощью Selenium с сайта CIAN, затем обрабатывает их при помощи библиотеки Matplotlib.

Структура проекта:

pars.py: скрипт для парсинга данных о стоимости квартир с сайта CIAN
clean_price.py: скрипт для очистки и обработки данных о стоимости квартир
gratp_price.py: скрипт для построения гистограммы цен
Технологии и библиотеки:

Selenium для парсинга данных
Matplotlib для построения гистограммы
Pandas для обработки данных
CSV для хранения данных
Как использовать:

Запустите скрипт pars.py для парсинга данных о стоимости квартир с сайта CIAN.
Запустите скрипт clean_price.py для очистки и обработки данных о стоимости квартир.
Запустите скрипт gratp_price.py для построения гистограммы цен.
